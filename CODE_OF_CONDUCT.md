# Code of Conduct

## Our pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Expected behavior

- Be respectful and constructive.
- Assume good intent, but focus on technical clarity.
- Accept feedback gracefully.
- Keep discussions on-topic and actionable.

## Unacceptable behavior

- Harassment, discrimination, or personal attacks.
- Trolling, insulting, or derogatory comments.
- Publishing private information without permission.
- Any conduct that is inappropriate in a professional setting.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct and may remove, edit, or reject comments, commits, code, issues, and other contributions that violate these standards.

## Reporting

If you experience or witness unacceptable behavior, report it through a private channel described in `SECURITY.md` or by contacting repository maintainers directly.
